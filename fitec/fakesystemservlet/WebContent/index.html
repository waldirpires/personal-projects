Welcome to the Fake System Servlet

The servlet sends back SOAP messages to RighTV and CDN adapters according to the request.

It is useful for testing workflows bypassing the nodes that depends on RighTV and CDN systems and they are not available.

Use the following URLs:

 - RighTV (all APIs and RTV sites in the "scac-adapter-rightv.properties" file): http://serverURL/WebServiceName/RTVServlet

 - CDN (connection string and heartbeat URLs in CMS UI): http://serverURL/WebServiceName/CDNServlet
